# NASA DE441 万年历

基于 NASA JPL 高精度行星星历与中国国家标准构建的专业级天文历法系统。

## 🌟 核心优势

### 1. 顶尖的天文学精度 (Astronomical Precision)
*   **DE441 星历驱动**：不同于市面上常见的基于简易公式（如 VSOP87）的日历，本系统由 NASA 的 **DE441 (Development Ephemeris)** 物理星历驱动，支持从公元前 13,000 年到公元 17,000 年的秒级位置计算。
*   **消除长周期漂移**：修正了**岁差 (Precession)** 导致的坐标系偏移。在计算万年尺度的节气时，明确使用“当日平春分点”坐标，确保公元 10,000 年的冬至准确锚定在 12 月 17 日，而非错误的 4 月。
*   **物理地球模型**：计算严谨考虑了**地球自转减速 ($\Delta T$ 效应)**。在万年尺度上，地球转速的变化会导致日历时间与力学时产生数日的偏差，本系统已完美对齐 NASA 的物理模型预报。

### 2. 严谨的中国历法逻辑 (Calendrical Robustness)
*   **GB/T 33661-2017 标准**：严格遵循中国国家标准《农历的编算和颁行》，精准定位冬至锚点与“无中气置闰”规则。
*   **双冬至锚点动态算法**：针对极端远期年份由于地球减速导致的“漏月”通病，系统采用了动态闭环搜索，确保三万年内的农历月份序列逻辑严密。
*   **完整传统细节**：
    *   **三柱干支**：实时推算年、月、日三柱干支。
    *   **五虎遁起月**：根据年干精准推算月柱天干（如甲辰年丁卯月）。
    *   **月首标注**：农历每月初一显示月份名，作为时间的自然分割。

### 3. 极致的性能与体验 (Extreme Performance)
*   **数据脱水压缩**：通过结构化元组存储技术，将 500 年全量天文数据（农历、节气、日月食）压缩至约 **800KB**，实现零延迟即时加载。
*   **极简主义美学**：
    *   **呼吸感排版**：12px 留白布局，中文化视觉语言。
    *   **智能指示器**：24 节气动态 Emoji 映射；背景中心衬托带银白色“日冕”发光效果的日月食水印。
*   **全场景适配**：完美支持移动端触控和 URL 参数年份分享。
*   **装裱级打印**：定制书法风格页眉与双边装饰线，支持 A4 尺寸、不跨页切断的实体年历打印。
